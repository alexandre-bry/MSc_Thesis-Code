# C++

## Dependencies

These libraries are used in the project:

- [PDAL (version 2.9.3)](https://pdal.org)
- [CLI11](https://github.com/CLIUtils/CLI11)
- [indicators](https://github.com/p-ranav/indicators)
- [CGAL (version 5.6.1)](https://www.cgal.org) (even though it is not really used yet)

[Dependencies.md](../Dependencies.md) gives instructions to install these libraries.
